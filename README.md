# vaner-docs

Public documentation site for [Vaner](https://vaner.ai), deployed at [docs.vaner.ai](https://docs.vaner.ai).

Built with [Fumadocs](https://fumadocs.vercel.app) and Next.js.

## Development

```bash
npm ci
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build check

```bash
npm run build
```

## Content

Documentation lives in `content/docs/`. Files are MDX and sidebar order is controlled by `meta.json` files.
